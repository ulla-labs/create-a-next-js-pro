# Next.js Modern Auth Template

A clean, modern authentication UI built with Next.js 14, Tailwind CSS, and Lucide React.

## Features
- 🔐 Login and Signup pages
- ✅ Basic form validation
- 📱 Fully responsive UI
- 🎨 Styled with Tailwind CSS
- 🧩 Reusable UI components

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure
- `src/app/login`: Login page logic and UI
- `src/app/signup`: Signup page logic and UI
- `src/components/ui`: Atomic components like Input
- `src/components/auth`: Layout wrappers for auth views
